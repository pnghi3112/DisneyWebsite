# Disney movies' values 
 This website explores the deeper messages woven into Disney’s animated classics and modern masterpieces—revealing how each story encourages us to believe in ourselves, embrace compassion, and find magic in the everyday.

## Overview

The Disney fansite project is a personal project that focus on telling people about how Disney spread their lessons in animaytions for young kids, hid in the fantasy and beatiful world in a clean, interactive, and user-friendly website.  

Objectives:   
- Introducing a few famous Disney animations 
- Have movie watching, interactive experiences for the audience 
- Develop and demonstrate web design and HTML development skills  

---

## Features

- Responsive layout compatible with desktop, tablet, and mobile devices  
- Interactive header through different sections of the website 
- Movies links with navigation to a movie site
- Clean and modern design with clear typography and layout  
- Easy maintenance and content updates
